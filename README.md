Desafio de Projeto DIO – Atualizando Relatório de Vendas com Foco na Experiência do Usuário
- Descrição do Projeto

Este projeto tem como objetivo aprimorar um relatório de vendas no Power BI, com foco na experiência do usuário (UX), organização visual e melhor interpretação dos dados. Foram aplicados conceitos de design e usabilidade para tornar o dashboard mais intuitivo, interativo e profissional.

O relatório é composto por 4 páginas, contendo visuais organizados estrategicamente, navegação entre páginas e melhoria na apresentação das informações.


- Objetivos

Melhorar a experiência do usuário no relatório

Organizar os visuais de forma mais clara e intuitiva

Aplicar conceitos de design como:

Posicionamento adequado dos elementos

Contraste visual

Proporção e alinhamento

Segmentação de dados

Criar botões de navegação entre páginas

Melhorar a estética e usabilidade do dashboard


- Ferramentas Utilizadas

Power BI Desktop

Modelagem de Dados

Visualizações interativas

Recursos de navegação (botões e menus)


- Principais Melhorias Implementadas

Reorganização dos visuais para melhor leitura

Ajuste de cores e contraste

Criação de matriz com descrição de vendas

Modificação e estilização dos gráficos

Implementação de botões de navegação

Criação de menus interativos entre páginas

Padronização visual do relatório


- Resultados Obtidos

O relatório tornou-se:

Mais intuitivo

Visualmente mais organizado

Mais profissional

Mais fácil de interpretar

Com melhor experiência para o usuário


- Aprendizados

Durante o desenvolvimento deste projeto, foram praticados:

Design de dashboards

Experiência do usuário (UX)

Storytelling com dados

Organização e padronização visual

Uso de botões e navegação no Power BI
